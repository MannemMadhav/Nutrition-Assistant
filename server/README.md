# 🥗 Nutrition Assistant AI

An AI-powered nutrition assistant built using the MERN Stack.

## Features

- JWT Authentication
- Meal Generator
- AI Nutrition Coach
- Dashboard Analytics
- Nutrition Tracking
- BMI & BMR Calculator
- Water Intake Tracker
- Profile Management
- Responsive Design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT
- Axios

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Environment Variables

Backend:

```
PORT=5000
MONGODB_URI=mongodb://127.0.0.1:27017/nutrition-assistant
JWT_SECRET=NutritionAssistant@2026#MadhavSecure123
OPENAI_API_KEY=your_openai_api_key
```

Frontend:

```
VITE_API_URL=http://localhost:5000/api
```
